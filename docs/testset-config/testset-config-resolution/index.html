    Browser Resolution

    Feb 15, 2023

    To change the browser resolution for your scenarios you have 3 choices.

    Changing the resolution for a specific scenario

    If you have a specific Scenario in which you want to run under a particular screen resoultion (perhaps you want to confirm a certain control behaves properly on tiny screens) then you can do this inline in your gherkin:

    When we resize the window to 100x100

    This will resize your window to 100 x 100 pixels. Please note: This will override all following settings for this scenario!

    Changing the default resolution for your TestSet

    If you wish all of your scenarios to run at a certain resolution you can configure this centrally. Create a RunConfiguration.json file as described in Run Configurations. then ensure that your active configuration has a resolution specified:

    "BrowserResolution": "100x100"

    This will resize the window to 100 x 100 pixels for every test. Note that if you specify the window size for a single test via gherkin then it will override this setting.
