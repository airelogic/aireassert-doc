    Browser Configuration

    Apr 18, 2023

    To change the browser for your scenarios you have several choices.
    When chosing a browser you have the choice of:

    • chromium
    • chrome
    • chrome-canary
    • internet explorer
    • edge
    • firefox
    • opera
    • safari

    PLEASE NOTE: You need the browser in question installed on your computer in order to use them to test with!

    Changing the default resolution for your TestSet

    If you wish all of your scenarios to run on a certain browser you can configure this centrally. Create a RunConfiguration.json file as described in Run Configurations. then ensure that your active configuration has a browser specified:

    "Browser": "chrome"
