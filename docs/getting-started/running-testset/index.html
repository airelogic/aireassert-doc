    Running a TestSet from the UI

    Jul 25, 2023

    Once you have written tests for your solution, you will want to run those tests.

    Before you begin

    • You should have created a TestSet and pushed it to a bitbucket git repository;
    • Your git repository should be connected to the AireAssert instance you will be using (This should be true as you will have been given a repository by adminstraitors wihthin the first few steps)

    in the tree: Select your testset in the tree and then click the Run Test Set button. Wait for the test started/queued message:

    Then move to the Test Run Results page. You should see your testset running. Wait for this to complete. (This may take some time)

    Now that the testrun is complete, we can inspect the results - click on the result and a dialog will open. From here you can find a list of tests which passed/failed, and you can dig right into the logs to investigate exactly what happened at each step.
